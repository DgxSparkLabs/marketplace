# Security Policy

This marketplace ships configuration plus example/reference plugins; the
repository itself executes no privileged code. Reference constructs that run
commands (the MCP/LSP examples) do so locally under the host tool's permission
model. Even so, if you find a security concern — a malicious payload path, a
construct that could exfiltrate data, or a supply-chain risk in an example's
upstream dependency — please report it **privately**.

## Reporting a vulnerability

Email the maintainer at **yorai.email@gmail.com** with a description and
reproduction steps. Please do **not** open a public issue for security reports.

You can expect an acknowledgement within a few days and updates as a fix is
prepared. Thank you for helping keep the marketplace and its users safe.
