# Example Rule — Always include units

When stating empirical measurements (latency, file size, memory usage, throughput, money), always include the unit. "Latency dropped" is unverifiable; "latency dropped from 240ms to 95ms" is reviewable.

This rule is a reference example. The body would normally be longer and more specific to your team's conventions. Keep rules action-oriented and short — they are read on every prompt.
